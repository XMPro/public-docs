# v4.4.18

## Common

{% hint style="info" %}
Security update: Mitigate a high-severity vulnerability by upgrading certain Agents.
{% endhint %}

<table><thead><tr><th width="157">Change Type</th><th>Description</th></tr></thead><tbody><tr><td>Security</td><td>We've replaced the deprecated System.Data.SqlClient library with Microsoft.Data.SqlClient, which is the recommended SQL Server data provider for all .NET applications. This update mitigates the risk of security vulnerabilities in the deprecated library while providing improved security, performance, and ongoing support. Refer to Integrations below for affected Agents.</td></tr></tbody></table>
